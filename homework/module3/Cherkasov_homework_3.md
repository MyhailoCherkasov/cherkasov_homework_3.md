## Завдання 2. Робота у фоні та керування процесами

### 2.1. Запустити довгу команду у фоновому режимі

Команда:
```bash
sleep 1000 &
```

Приклад виконання:
```text
Запущено sleep 1000 у фоні. PID: 2022
```

### 2.2. Перевірити список фонових завдань

Команда:
```bash
jobs -l
```

Приклад результату:
```text
[1]+  2022 Running                 sleep 1000 &
```

### 2.3. Повернути процес із фону на передній план

Команда:
```bash
fg %1
```

Після повернення процесу на передній план його можна зупинити комбінацією клавіш:
```text
Ctrl + Z
```

### 2.4. Примусово завершити процес за допомогою kill

Команда:
```bash
kill -9 <PID>
```

Приклад виконання:
```text
Процес 2022 завершено командою kill -9
```

### 2.5. Запустити команду через nohup

Команда:
```bash
nohup sleep 1000 > nohup_sleep.log 2>&1 &
```

Приклад виконання:
```text
nohup sleep 1000 запущено. PID: 2023
  PID  PPID STAT COMMAND         COMMAND
 2023  2013 S    sleep           sleep 1000
```

## Завдання 3. Пріоритети та обмеження
